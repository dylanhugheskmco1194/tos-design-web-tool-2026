## What TOS Design Is

TOS Design is a web-first design workspace centered on HTML, giving you a way to open, edit, and arrange visual content without relying on a standalone desktop app. It is intended for people who want a compact browser tool for design-oriented tasks and a workflow that stays easy to access.

Since the project lives in the browser, it suits environments where fast availability is important and where a normal web browser is the most practical runtime. Its HTML-based structure also makes hosting, opening, and adapting the tool straightforward in a typical web setup.
